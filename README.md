# JEFE Landing Page

This is the landing page for JEFE, a platform connecting foremen with industrial companies.

## Tech Stack
- React
- Vite
- Tailwind CSS v4
- Lucide React (Icons)

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
